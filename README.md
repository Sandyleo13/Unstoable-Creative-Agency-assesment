# Product Listing - Assessment

This is a ready starter project for the Full Stack Developer assessment.
Features:
- React + Vite starter
- Fetches products from https://dummyjson.com/products
- Responsive cards, loading state, error state, search filter

To run locally:
1. Install Node.js 20+ (recommended)
2. npm install
3. npm run dev
